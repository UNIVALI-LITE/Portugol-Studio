Este pacote contém tradutores para todos os tipos de erros de parsing gerados pelo ANTLR durante a análise sintática.

Os tradutores são responsáveis por capturar os erros de parsing gerados pelo ANTLR, os quais possuem mensagens genéricas (e em inglês), e traduzí-los para erros sintáticos com mensagens mais específicas e detalhadas (e em Português).
