Desvio Condicional



Nesta sessão encontram-se exemplos dos comandos "se" e "senao", os usados para fazer comparações entre dados:

Exemplo 1

Algoritmo que verifica se o numero é par ou impar 



programa 
    {
      funcao inicio (cadeia parametros[])
         {

	inteiro a
	escreva ("Digite um numero ")
	leia (a)

	se (a % 2 == 0) {
           escreva ("\nO numero é par")
	} senao {
           escreva ("\nO numero é Impar")
	}

    }
}


Exemplo 2

Algoritmo que verifica qual foi o maior numero digitado 



programa 
     {
	funcao inicio (cadeia parametros[])
	   {

 	inteiro a, b
	escreva ("Digite um numero ")
	leia (a)
	escreva("\nDigite outro numero ")
	leia (b)

	se (a > b) {
	   escreva ("\n",a," é maior que ",b) 
	} senao se (a < b) {
	   escreva ("\n",b," é maior que ",a)
	} senao {
	   escreva ("\nOs dois são iguais")
	}

     }
}


Exemplo 3

Algoritmo que realiza ou a operação de divisão ou multiplicação 



programa 
      {
        funcao inicio (cadeia parametros[])
	   {

	inteiro a, b
	real resultado

	escreva ("Digite um numero ")
	leia (a)
	escreva ("\nDigite 1 para metade do numero digitado ou 2 para o dobro do numero digitado")
	leia (b)

	escolha (b) {
	caso 1:
	  resultado = a / 2
	pare
	caso 2:
	  resultado = a * 2
	pare
        }

	escreva ("\n\nO resultado é ",resultado)
     }
}
