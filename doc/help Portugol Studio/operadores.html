Operadores

  No Portugol Studio são utilizados os seguintes Operadores:

Operadores Aritméticos

  + (soma)
  - (subtração)
  / (divisão)
  % (resto divisão)
  * (multiplicação)

Operadores Relacionais

  > (maior)
  < (menor)
  > = (maior igual)
  < = (menor igual)
  = = (igual)
  != (diferente)

Operadores Lógicos

  e (operador E)
  ou (operador OU)
  não (operador NAO)

Operadores

  ++ (incremento)
  -- (decremento)
  += (adição com atribuição)
  -= (subtração com atribuição)
  /= (divisão com atribuição)

Exemplo 1:

  programa
  {
       funcao inicio( )
       {
          real a, b, c, d, f

          a = 3 + 8 * 2
          b = 40 - 24 / 3 * 4
          c = 51 / 4
          d = 39 % 5
          escreva (a,"\n")
          escreva (b,"\n")
          escreva (c,"\n")
          escreva (d,"\n")
       }
  }

Exemplo 2:

  programa
  {
      funcao inicio ( )
      {
         inteiro a,b,c,d,f,x

         b = 8 * 2 - 5
         c = 15 - 25
         d = 6
         leia (a)
         leia (f)

         b = b % 5
         c = c + 12
         d = d * 5 + 2
         x = a + f * c - d
         escreva (x)
      }
  }
