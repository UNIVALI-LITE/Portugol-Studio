Comando escreva

Tópicos de Ajuda >> Informaçãos de Uso >> Comandos Básicos >> Comanto escreva

O que é?

  O comando ESCREVA é utilizado quando deseja-se mostrar informações na tela do computador, ou seja, é um comando de saída de dados. Esses dados podem ser valores de variaveis e informações para usuário na tela.

Como usar?

  Para utilizar o comando ESCREVA, você deverá escrever este comando e entre parênteses colocar a(s) variavel (eis) que você quer mostrar na tela. Quando você utilizar textos com variaves, o texto deve estar entre aspas. como utlizar o comando ESCREVA:

COMANDO ESCREVA
Formas de Utilizar Exemplo Descrição Resultado
1 escreva (a) escreve o valor da variavel a na tela
2 escreva (a,b,c,d,f) escreve o valor das variaveis a,b,c,d,f na tela
3 escreva ("valor"; a) Escreve a palavra valor e o valor da variavel a na tela.

Veja os exemplos a seguir:

Exemplo 1:

  programa
  {
       funcao inicio ( )
       {
          inteiro idade

          escreva ("Digite sua idade")
          leia (idade)

          escreva ("você tem", idade , "anos de idade")
       }
  }

Exemplo 2:

  programa
  {
       funcao inicio( )
       {
          inteiro dobro, triplo
          real metade, valor

          escreva ("digite um valor \n ")
          leia (valor)

          metade = valor / 2
          dobro = valor * 2
          triplo = valor * 3

          escreva ("A metade do número é: ",metade,"\n")
          escreva ("O dobro do número é: ",dobro,"\n")
          escreva ("O triplo do número é: ",triplo,"\n")
       }
  }
