O que é?

  Um vetor pode ser visto como uma variavel que possui diversas posições, e com isso armazena diversos valores, porem todos do mesmo tipo.

Como usar?

  Na declaração de vetor, indica-se o tamanho destes. Os índices devem aparecer entre conchetes. Caso o usuário não informe o índice, o programa vai considerar como sendo zero.

VETOR
Formas de Utilizar Exemplo Descrição
1 < tipo > vetNome[tamanho] Sintaxe
2 inteiro vetValores[5] = {1,2,3} Declaração de vetor com inicialização
3 const inteiro vetValores[ ] = {1,2,3,4,5,6} Declaração de vetor com constante.
